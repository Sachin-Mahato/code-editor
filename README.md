## Installation

### Prerequisites

- [Node.js](https://nodejs.org/) (v18+ recommended)
- [pnpm](https://pnpm.io/) or [npm](https://www.npmjs.com/)

### Steps

```sh
pnpm install
# or
npm install
```

## Usage

```sh
pnpm dev
# or
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Running Tests

```sh
pnpm test
# or
npm run test
```
